Arquitetura do Computador

A Arquitetura de Computadores é o projeto conceitual e fundamental da estrutura operacional de um sistema computacional. Ela é o estudo dos requisitos necessários para que um computador funcione e de como organizar os diversos componentes para obter melhores desempenhos.

Mas o que acontece quando ligamos o computador?

Todo mundo que utiliza um computador alguma vez já teve a dúvida sobre o que acontece quando apertamos o botão de ligar. Por isso fizemos um resumo logo abaixo para responder essa questão.

1 - Ao apertar o botão ligar.

A energia começa a fluir dentro do computador.

2 - A CPU

O primeiro a acordar é a cpu e ela vai buscar o que vai fazer com a BIOS ou UEFI.

3 - A BIOS

Quando a BIOS é iniciada ela vai acordar os componentes de acordo com uma configuração predeterminada e executar o POST (Power On Self Test) que verifica se os componentes estão funcionando adequadamente, então depois desse processo ele vai buscar o sistema operacional na memória secundária (hd ou ssd). Curiosidade: também dá pra iniciar o sistema operacional pelo pendrive.

4 - O Bootloader

Em seguida o bootloader(carregador de inicialização) que fica na MBR (Master Boot Record) do disco rígido ele carrega o sistema operacional. dependendo do sistema esse bootloader pode ser o GRUB ou LILO para linux, boot.efi para mac e windows boot manager para windows.

5 - E o que vem depois da BIOS?

Depois o sistema operacional no hd é jogado na memória principal(memória ram).

6 - A última etapa: a inicialização do sistema operacional

E quando o sistema operacional está na memória principal, já dá para ver o windows na tela.


Curiosidade: com um hd todo esse processo é feito em média em 1m e 39s, com um ssd apenas 22s. Lembrando que os tempos podem variar de acordo com o sistema e a máquina utilizada.
